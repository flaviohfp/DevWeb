Banco de Dados de Criaturas Paranormais

Após meses de silêncio, a Ordem volta a entrar em contato. Algo deu errado.

O Sistema de Registro de Criaturas, essencial para catalogar e entender ameaças paranormais, foi parcialmente destruído. Sem esse sistema funcionando, a humanidade fica às cegas, incapaz de identificar, estudar ou se proteger das criaturas que surgem nas sombras.

Agora, novos agentes são convocados. Antes de qualquer confronto direto com o paranormal, a missão é clara: reconstruir o sistema, recuperar informações perdidas e organizar os registros novamente. Só assim será possível compreender o que está lá fora — e sobreviver.
