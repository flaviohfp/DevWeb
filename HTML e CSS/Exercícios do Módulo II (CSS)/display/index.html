A Propriedade de 'display' no CSS

Abaixo. veja com os elementos se comportam dependendo do seu valor de 'display'

1. Display: block

Elementos 'block' ocupam toda a largura e sempre forcam uma quebra de linha, mesmo que haja espaco sobrando ao lado.

2. Display inline-block

Elementos 'inline' so o ocupam o espaco do seu conteudo e ficam lado a lado. Eles **Ignoram** as definicoes fr 'width' e 'height'.

inline A inline B (Com conteudo Maior) Inline C

3.Display: Inline-Block

O melhor dos dois mundos: ficam lado a lado E respeitam as definicoes de 'width' e 'height'.
